# Java Full Stack Foundation

An Advanced Java full-stack learning project built using Spring Boot, JavaFX, MySQL, Hibernate (JPA), and REST APIs.

This project is focused on understanding real-world full-stack software architecture using Java technologies.

---

# Project Architecture

JavaFX Frontend  
↓  
REST API Communication  
↓  
Spring Boot Backend  
↓  
Service Layer  
↓  
Repository Layer  
↓  
Hibernate / JPA  
↓  
MySQL Database

---

# Current Features

- REST API development using Spring Boot
- CRUD operations for Student entity
- MySQL database integration
- JavaFX desktop frontend
- Frontend-backend communication using HTTP requests
- JSON request handling
- Git and GitHub integration
